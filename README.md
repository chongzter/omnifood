# omnifood
Omnifood is a premium food delivery service with the mission to bring affordable and healthy meals to as many people as possible.

## Description
This website is build on these technologies:
- HTML5
- CSS
- JS
- JQuery

Libraries use:
- Normalize CSS template
- Grid CSS template
- Ionicons
- Google Fonts
- Respond JS
- html5shiv
- selectivizr
